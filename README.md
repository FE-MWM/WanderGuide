## 개발환경 세팅 & 실행

1. 클론한 프로젝트를 npm install 해줍니다.
2. npm run build 명령어를 실행해줍니다.
3. 크롬 브라우저의 '확장프로그램 관리' 페이지로 이동합니다
4. 좌측 상단의 '압축 해제된 프로그램을 로드합니다' 클릭
5. 해당 프로젝트의 build 폴더를 선택합니다
6. 로드된 크롬 확장 프로그램 아이콘을 클릭합니다
